Password management Tools

Browse open-source password management tools. These alternatives help you replace proprietary software while keeping control of your data.

  • Bitwarden

    Bitwarden is a popular open-source password manager for individuals and teams.

    Alternative to: 1Password

  • KeePass

    KeePass is a classic open-source password manager focused on local storage.

    Alternative to: LastPass

  • Padloc

    Padloc is a modern open-source password manager with a clean interface.

    Alternative to: 1Password

  • Passbolt

    Passbolt is an open-source password manager built for teams.

    Alternative to: 1Password

  • Vaultwarden

    Vaultwarden is a lightweight, self-hosted Bitwarden-compatible server.

    Alternative to: Bitwarden
