Web analytics Tools

Browse open-source web analytics tools. These alternatives help you replace proprietary software while keeping control of your data.

    Alternative to: Google Analytics

    Alternative to: Google Analytics

    Plausible is a simple, privacy-friendly open-source analytics tool.

    Umami is a lightweight, self-hosted open-source analytics platform.
